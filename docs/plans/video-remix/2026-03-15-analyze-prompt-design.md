# Analyze Prompt Design

**Goal:** 调整 `video-remix` 的分析提示词，让分析结果以中文输出，并覆盖完整原文分段文稿、核心论点、金字塔结构拆解、反对者视角、传播营销视角等内容。

## Scope

- 仅修改 `skills/video-remix/resources/prompts/analyze.md`
- 保持 `stages/analyze.py` 的调用方式不变
- 增加一个测试，约束分析提示词中的关键章节，防止后续回退

## Output Design

分析结果固定使用中文 Markdown，包含以下章节：

- `# 分析`
- `## 原文的分段文稿（完整）`
- `## 核心论点`
- `## 金字塔结构拆解`
- `## 关键论据与例子`
- `## 反对者视角`
- `## 传播营销视角`
- `## 受众与表达意图`
- `## 风险与歧义`

## Prompt Intent

- “原文的分段文稿（完整）”要求按原意重组为完整中文文稿，不允许只给摘要或提纲
- “金字塔结构拆解”要求区分总论点、一级支撑、二级支撑
- “反对者视角”要求给出最强反方攻击点，而不是泛泛而谈
- “传播营销视角”要求提炼传播钩子、情绪点、标题方向和二次改写切口

## Testing

- 新增针对 `analyze.md` 文本内容的测试
- 断言关键中文章节存在，确保本次提示词目标被保留
